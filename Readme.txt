Url Shortener is a tool that can be used to shorten your lenghty urls and links.

How to run this:
**Make sure you have PIP and Python 3 already installed on your system**

1: Open Command Prompt
2:Run "pip install pyshortener"
3:Let it completely install.
4:Open any text Editor(Sublime,Python IDLE,etc) 
5:Paste the above code and save it in a folder.
6:Now open Command prompt in that folder.
7:Type "python *name of the python file*" and press Enter.
